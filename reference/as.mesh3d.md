# Convert a raw neuroglancer mesh into a mesh3d object

Convert a raw neuroglancer mesh into a mesh3d object

## Usage

``` r
# S3 method for class 'ng_raw'
as.mesh3d(x, ...)

# S3 method for class 'ng_raw_list'
as.mesh3d(x, ...)
```

## Arguments

- x:

  An ng_raw or ng_raw_list object

- ...:

  additional arguments passed to `tmesh3d`
