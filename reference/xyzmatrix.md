# Extract the XYZ coordinates from neuroglancer mesh objects

Extract the XYZ coordinates from neuroglancer mesh objects

## Usage

``` r
# S3 method for class 'ng_raw'
xyzmatrix(x, ...)

# S3 method for class 'ng_raw_list'
xyzmatrix(x, ...)
```

## Arguments

- x:

  an ng_raw or ng_raw_list object

- ...:

  additional arguments (currently ignored)
