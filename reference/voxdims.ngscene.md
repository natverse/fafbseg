# Extract voxel dimensions from a neuroglancer scene

Extract voxel dimensions from a neuroglancer scene

## Usage

``` r
# S3 method for class 'ngscene'
voxdims(x, units = c("nm", "um", "microns", "mm", "m"), ...)
```

## Arguments

- x:

  A neuroglancer scene as generated by
  [`ngl_decode_scene`](https://natverse.org/fafbseg/reference/ngl_decode_scene.md).

- units:

  Optionally specify output units for the voxel dimensions. `NA` uses
  the original units.

- ...:

  additional arguments (currently ignored)
